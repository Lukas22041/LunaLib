This folder holds additional libraries. Those can be added as Libraries to your own mod, and aslong as Lunalib is loaded, those will be loaded aswell.
Details for each Library are below.

fuzzywuzzy-1.3.0.jar
A library for fuzzy string matching. If you want text input to be recognized despite strings not perfectly matching, you can use this library for it.
Details on its github: https://github.com/xdrop/fuzzywuzzy
